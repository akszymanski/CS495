Visit the Github Repo

Project Overview

This project is meant to serve as both a portal for managing building certification inspections and a mobile UI that will record the field-inspectors responses to pre-defined certification standards and generate a building certification report after processing the inspection responses through the certification API.

Project Workflow

Frontend workflow

To work on the frontend navigate to the root directory of the project called "field-inspector-portal" then /apps/native to work on the mobile side. To work on the web side go to /apps/web.

Backend workflow

To work on the backend navigate to the root directory of the project called "field-inspector-portal" then /backend/ to work on the backend/ and the folder structure side.
